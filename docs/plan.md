# dev-notes-cli Plan

Version 0.1 focuses on one thing: generate clean, consistent markdown notes for technical writing.

## Scope

- `new`: create markdown file with front matter + template body
- `init`: generate `.devnotes.yaml`
- `list-templates`: show built-in templates
- `doctor`: validate config and output directory

## Built-in templates

- `note`
- `project`
- `interview`
